**Goal:** Friendly presets while keeping low-level flags.

**Frontend**

* `forge3d.presets`:

  * `studio_pbr()`: dir light + IBL GGX + PCF shadows + Disney.
  * `outdoor_sun()`: Hosek sky + sun (directional) + CSM + GGX.
  * `toon_viz()`: toon BRDF + hard shadows + no GI.
* `Renderer.apply_preset(name: str, **overrides)` merges into current config.

**Examples**

* `examples/lighting_gallery.py`: grid render swapping BRDFs/lights.
* `examples/shadow_gallery.py`: compare techniques side-by-side.
* `examples/ibl_gallery.py`: HDR rotation and roughness sweep.

**Docs**

* Sphinx how-to pages for each preset; CLI snippets.

**Acceptance**

* Users can reproduce a figure with one command:

  ```
  python examples/terrain_demo.py --preset outdoor_sun \
    --brdf cooktorrance-ggx --shadows csm --cascades 4 --hdr assets/sky.hdr
  ```

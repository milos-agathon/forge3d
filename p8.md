# P8 — Performance & memory budget enforcement (≤512 MiB host-visible)

**Goal:** Guardrails + telemetry.

**Backend**

* Central allocator reports live bytes by class (vertex, index, textures by format, shadow maps, IBL).
* Hard caps: shadow atlas, froxel grid, prefilter sizes; refuse/scale with warning.
* CLI `--gpu-budget-mib 512` default; auto-downscale atlas or IBL res to fit.

**Frontend**

* `renderer.get_stats()` returns dict (CPU/GPU mem, frame time, passes enabled).

**Acceptance**

* For tight budget, app prints decisions: “Shadow atlas reduced to 1024² to fit budget”.
